# PyML 

**Author:** Vaibhav Arun Patil  
**Version:** 0.1.0  

PyML is a complete Data Science, Machine Learning, and NLP framework designed to simplify end-to-end AI workflows with powerful automation and intelligent recommendations.

---

##  Features

###  Data Preprocessing
- Missing value detection
- Smart imputation suggestions
- Encoding (Label, One-Hot)
- Feature scaling (Standard, MinMax)
- Normalization
- Dimensionality Reduction (PCA)
- Imbalanced dataset handling

###  Machine Learning
Supports:

#### Regression Models
- Linear Regression
- Ridge
- Lasso
- Random Forest Regressor
- SVR

#### Classification Models
- Logistic Regression
- Decision Tree
- Random Forest
- SVM
- KNN
- Naive Bayes

#### Clustering Models
- KMeans
- DBSCAN

---

###  AutoML Intelligence
- Automatic problem type detection
- Smart preprocessing suggestions
- Automatic model comparison
- Best model selection
- Model ranking system

---

###  Model Explainability
- Feature importance extraction
- Linear model coefficients
- Transparent AI support

---

###  Advanced Visualization
- Histogram
- Box Plot
- Violin Plot
- Scatter Plot
- Correlation Heatmap
- Pairplot
- Interactive Plotly charts
- Confusion Matrix
- Feature Importance Plot

---

###  Reporting System
- Dataset summary
- Missing value report
- Target distribution
- Automatic CSV report generation
